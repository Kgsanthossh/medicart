 MediCart – Online Medical Store

MediCart is a **React.js-based online pharmacy web application** that allows users to browse, search, and purchase medicines and healthcare products with ease.  
It focuses on delivering a **smooth user experience**, responsive design, and essential e‑commerce features like cart management and checkout.



 Features
- 🔍 **Medicine Search & Filter** – Quickly find medicines by name or category  
- 🛒 **Cart Management** – Add, update, or remove products  
- 📦 **Order Tracking** – View order status and history  
- 💳 **Secure Checkout** – Simple and safe checkout process  
- 📱 **Responsive Design** – Works seamlessly on all devices  



 Tech Stack
- **Frontend:** React.js, Tailwind CSS  
- **State Management:** React Hooks / Context API  
- **Backend (optional):** Node.js + Express  
- **Database (optional):** MongoDB / Firebase  
- **Deployment:** Vercel / Netlify
